Архив содержит сервис для получения информации о парах курсов валют.

Для развертывания сервиса, необходимо выполнить следующее:
- установить Docker
- разархивировать exchanges_epayments.zip в exchanges_epayments
- из консоли
	- перейти в папку с исходным кодом ~\exchanges_epayments
	- вызвать команду docker-compose up
- перейти в браузере по адресу http://localhost/swagger
- страница содержит документацию по работе с API сервиса
- для проверки работоспособности сервиса можно перейти в браузере по адресу http://localhost/api/ping

Описание архитектуры:
Для работы сервиса используется несколько компонентов:

- WebAPI:
	получение информации о парах курсов валют 
	>> для получения информации необходимо использовать маршрут: /api/rates
- Quartz
	планировщик для запуска процедур загрузки информации курсов валют (
	>> для выбора провайдера, необходимо открыть файл appsettings.json и выставить флаг IsEnabled
	у одного из провайдеров данных в секции Providers, а также период инвалидации данных в миллисекундах
- Mongo:
	NoSQL БД для хранения информации загружаемой от провайдеров информации
	>> БД разворачивается автоматически при запуске команды docker-compose up
- Redis:
	Key-Value БД для кэширования информации загружаемой от провайдеров в течении периода инвалидации
	>> БД разворачивается автоматически при запуске команды docker-compose up
- Swagger:
	документация WebAPI
